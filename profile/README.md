# Cacique Telecom

Proveedor de internet por fibra óptica y radiofrecuencia en La Victoria, San Mateo y Zuata, Venezuela. Planes desde 250 Mbps hasta 600 Mbps con soporte técnico garantizado.

## Proyectos

| Repositorio | Descripción |
|-------------|-------------|
| [PagosCCQ](https://github.com/caciquetelecom/PagosCCQ) | Portal de pagos para clientes — integración BNC, Wisphub y HKA |

## Stack

PHP 8.3 · Laravel 12 · MySQL · Alpine.js · Tailwind CSS
